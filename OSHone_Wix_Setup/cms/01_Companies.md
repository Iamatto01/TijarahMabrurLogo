# Collection Name: `Companies`
*Collection ID: `Companies`*

Pastikan set **Permissions** kepada "Site Members" untuk View, dan "Admin" untuk Update/Delete (atau ikut kesesuaian).

| Field Name | Field Key (Wix Auto) | Field Type |
| :--- | :--- | :--- |
| Company Name (Primary) | `title` | Text |
| SSM No | `ssmNo` | Text |
| Plan | `plan` | Text (Isi: "Basic" / "Pro") |
| Owner | `owner` | Reference (Link ke **Members/PrivateMembersData**) |
